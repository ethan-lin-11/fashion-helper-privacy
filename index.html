# AI 穿搭小幫手 - 隱私權政策 ## 資料收集與使用 本擴充功能「AI 穿搭小幫手」致力保護用戶隱私。 ### 我們收集的資料： • **網站內容**：用戶截圖的服飾商品圖片和上傳的個人照片 • **用途**：僅用於 AI 穿搭合成功能 • **保存期間**：處理完成後立即刪除，不會儲存在我們的伺服器 ### 我們不會： • 儲存您的個人照片 • 分享您的資料給第三方 • 追蹤您的瀏覽記錄 • 收集個人身分資訊 ### 資料處理： • 所有圖片處理透過安全的雲端 AI 服務 • 處理完成後立即刪除所有圖片資料 • 不會建立用戶資料庫或個人檔案 ### 權限用途： • `activeTab`：截取當前網頁商品圖片 • `storage`：暫存設定，提升使用體驗 • `desktopCapture`：螢幕截圖功能 • `scripting`：在網頁中註入截圖工具 • `host_permissions`：支援所有購物網站截圖功能 ### 聯絡方式： 如有隱私相關疑問，請透過 GitHub Issues 聯繫。 最後更新：2024年8月29日
